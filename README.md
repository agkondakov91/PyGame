# 🎮 Pygame Runner — учебный курс

Курс по разработке 2D runner-игры на Pygame. Студенты шаг за шагом создают игру с нуля: персонаж бежит, прыгает, собирает монеты, избегает препятствий. Параллельно осваиваются базовые принципы 2D-разработки: игровой цикл, анимация, столкновения, состояния игры, архитектура проекта.

---

| # | Тема                                                                         |

## 🚀 Быстрый старт

### 1. Клонировать репозиторий

```bash
git clone git@github.com:agkondakov91/PyGame.git
cd PyGame
```

### 2. Проверить uv

```bash
uv --version
```

Если не установлен:

```bash
# macOS / Linux
curl -LsSf https://astral.sh/uv/install.sh | sh

# Windows
powershell -c "irm https://astral.sh/uv/install.ps1 | iex"
```

### 3. Установить Python 3.13

```bash
uv python install 3.13
uv python pin 3.13
```

### 4. Установить зависимости

```bash
uv sync
```

Если `pygame` не устанавливается, используйте `pygame-ce`:

```bash
uv remove pygame
uv add pygame-ce
```

Импорт в коде остаётся тем же: `import pygame`.

### 5. Запустить игру

```bash
uv run python main.py
```

## 🐞 Частые проблемы

**`ModuleNotFoundError: No module named 'pygame'`**
Запускайте только через `uv`:
```bash
uv run python main.py  # ✅
python main.py         # ❌
```

**`uv python pin 3.13` ругается**
Проверьте `pyproject.toml`. Должно быть:
```toml
requires-python = ">=3.13,<3.14"
```

**Картинка не загружается**
Убедитесь, что файл лежит в `assets/images/` и имя совпадает с кодом. Запускать из корня проекта.

**Столкновение происходит слишком рано**
Используйте хитбоксы: 
```python
player_rect.inflate(-40, -20)
```
